# Анализ выигрышных стратегий в игре "Камешки"

## Условие задачи

Рассмотрим упрощенный вариант игры. Дана **одна** кучка, в которой находится `n` камней. Два игрока по очереди делают ходы. За один ход игрок может убрать из кучки **1, 2 или 3 камня**. Выигрывает тот игрок, который забирает последний камень.

**Задача:** Для заданного начального количества камней `n` определить, у какого игрока есть выигрышная стратегия при оптимальной игре обоих игроков.

## Требования

1. Изучить теоретические основы игры "Камешки"
2. Доказать критерий выигрышности позиции
3. Разработать алгоритм определения выигрышной стратегии
4. Привести примеры работы алгоритма

## Формат решения

- Математическое доказательство теоремы
- Алгоритм определения выигрышной стратегии
- Примеры для различных начальных значений `n`

---

### Особенности этой задачи:

- **Проще оригинальной** - только одна кучка вместо нескольких
- **Ограниченные ходы** - можно брать только 1, 2 или 3 камня (вместо любого количества из любой кучки)
- **Легче для анализа** - простая закономерность выигрышных/проигрышных позиций
- **Хороша для начинающих** - позволяет понять основы теории игр на простом примере

Эта задача отлично подходит для первого знакомства с комбинаторными играми и выигрышными стратегиями.
